This is a simple server for Over-the-Air (OTA) provisioning of so-called MIDlet suites, that is, small Java applications conforming to the Mobile Information Device Profile (MIDP) that can be run on a variety of Java-powered mobile phones. The following suites are available for download. Click on any of the names to initiate the transfer. If you are accessing this page from a mobile phone, you might want to use the WAP version instead.

The OTA server is part of the Antenna project. Antenna provides an Ant-based open-source solution for the development and deployment of MIDlet suites. The project is hosted on SourceForge.
